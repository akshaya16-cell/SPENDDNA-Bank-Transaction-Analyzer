# 💳 SpendDNA – Bank Transaction Analyzer

## 📌 Project Overview

SpendDNA is a Python-based data analytics project that analyzes six months of bank transactions to uncover spending habits, financial trends, and unusual transactions. The project cleans raw transaction data, extracts merchants, categorizes expenses, performs statistical analysis, and generates insights into personal spending behavior.

---

## 🚀 Features

* Transaction data cleaning and preprocessing
* Vendor extraction using keyword mapping
* Automatic expense categorization
* Spending overview (Credits, Debits, Savings)
* Monthly spending trend analysis
* Time-based spending analysis
* Anomaly detection using Z-Score
* Spending archetype detection
* Final financial summary report

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Google Colab

---

## 📂 Dataset

The dataset contains six months of bank transactions with details such as:

* Date
* Description
* Amount
* Transaction Type
* Vendor
* Category

## ▶️ How to Run

1. Clone the repository.
2. Install the required libraries:

```
pip install -r requirements.txt
```

3. Open `SpendDNA.ipynb`.
4. Run all cells in sequence.
